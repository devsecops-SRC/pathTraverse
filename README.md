#### pathTraverse.py
功能: 批量检测目录遍历

安装依赖: `pip install requests threadpool`

Usage: python pathTraverse domain.txt \[path.txt\]

domain.txt: 需要检测的域名

path.txt: 检测所使用的字典,默认使用path_fast.txt

实例: python pathTraverse url.txt
